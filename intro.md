# Mathematical Basics for Computational Neuroscience

Welcome to the seminar on **Mathematical Basics for Computational Neuroscience**.

This course bridges the gap between mathematical foundations and modern neuroscience methods. We start from the basics of matrix theory and build toward the dimensionality reduction and data analysis techniques that drive current neuroscience research.

## Who This Course Is For

This seminar is designed for a **mixed audience**: neuroscientists who want to understand the math behind the tools they use, and mathematicians/engineers who want to see how linear algebra powers modern neuroscience. Each topic includes:

- **Formal definitions and theorems** for mathematical rigor
- **Intuition blocks** that build geometric and conceptual understanding
- **Neuroscience connections** that link the math to real applications and papers
- **Interactive demos** to visualize concepts
