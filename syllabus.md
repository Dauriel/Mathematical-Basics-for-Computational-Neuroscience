# Syllabus

## Course Overview

This seminar explores linear algebra and its applications in computational neuroscience. The course is organized into five modules that progress from mathematical foundations to advanced analysis techniques used in modern neuroscience research.

## Modules

| Module | Topics | Approx. Sessions |
|--------|--------|-------------------|
| 1. Foundations | Matrices, properties, tensors, change of basis | |
| 2. Eigentheory & Decompositions | Eigenvalues/vectors, dynamical systems, SVD, spectral analysis | |
| 3. Linear Dimensionality Reduction | PCA, LDA, cPCA, jPCA, dPCA, TCA, CCA, ICA, FA, MDS | |
| 4. Non-Linear Methods & Clustering | UMAP, CEBRA, K-Means, DBSCAN | |
| 5. Geometry & Topology | Neural manifolds, method selection, integration | |

## Format

Each session may include:
- **Lecture**: Instructor-led presentation of theory and methods
- **Paper discussion**: Group discussion of a key paper
- **Interactive demo**: Hands-on exploration of concepts via notebooks
- **Student presentations**: In later modules, students present methods applied to data

## Prerequisites

- Basic calculus (derivatives, integrals)
- Some exposure to linear algebra (vectors, matrices) is helpful but not required
- No programming experience required to follow the lectures; Python knowledge helpful for demos
